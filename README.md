Install project with npm

```bash
  git clone https://github.com/gregory-pet-projects/poc-mobile.git
  cd poc-mobile
  npm i
```

Add .env file
```bash
  API_BASE_URL=http://localhost:3300/api
```
Run project on ios simulator
```bash
  npm run ion
```
Run project on android simulator
```bash
  npm run android
```
